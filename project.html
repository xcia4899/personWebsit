Horizon ｜電商網站

此專案為個人從 0 開始規劃與設計之電商平台，並將既有 Vue 2 架構重構為 Nuxt 3，導入 TypeScript 與 Supabase， 建立具備型別安全與可維護性的應用架構。

專案涵蓋商品展示、搜尋篩選、購物流程與後台 CRUD 管理， 透過 RESTful API 串接前後端，實作使用者驗證與權限控管， 並整合狀態管理與表單驗證，提升系統穩定性與使用者體驗。

專案特色

  • 重構既有 Vue 2 專案為 Nuxt 3 架構，導入 TypeScript，提升型別安全與可維護性。
  • 使用 Pinia 建立全域狀態管理，統一資料流與跨元件狀態共享。
  • 串接 Supabase（Auth + Database），完成會員登入、註冊與資料存取機制。
  • 建立商品 CRUD 功能，完成前後端資料操作流程。
  • 實作商品搜尋、分類篩選、購物車與商品詳細頁功能。
  • 建立登入驗證與路由權限控管，限制後台管理頁面存取。
  • 使用 Element Plus 建立 UI 元件與表單系統，提高開發效率與介面一致性。
  • 實作表單驗證、錯誤提示與例外處理，提升使用者體驗。
  • 部署於 Vercel，建立自動化部署流程與環境管理。
